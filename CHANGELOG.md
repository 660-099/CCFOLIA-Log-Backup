# Changelog

## [1.1.1] - 2026-04-18
- 설정 기억하기(LocalStorage) 토글 스위치 및 브라우저 저장소 연동 기능 추가 
- 다양한 구조의 HTML 로그 지원을 위해 파싱(parser.ts) 로직 유연화 (모든 태그와 color 속성 탐지)
- 파일 업로드 UI 직관성 개선 (업로드 파일명 표시, 즉각적인 컬러 하이라이팅, X 취소 기능 추가)
- 화이트 테마에서 일부 안내 텍스트가 안 보이던 UI 버그 수정 및 파일 업로드 심볼의 애니메이션(Pulse) 제거
- (이전 버전 누락 내역: 호버 툴팁이 빠르게 사라지던 현상을 투명 가상 영역으로 수정)

## [1.1.0] - 2026-04-16
- 미리보기 화면 UI 개선 (섹션 이동 버튼 위치 및 애니메이션 조정, 검색 버튼 스타일 동기화, 첫 번째 섹션 바로가기 스크롤 버그 수정)
- 미리보기 창 검색 및 필터링 기능 추가, 탭 기본 색상 변경(#ffd400) 및 잡담 탭 기본 활성화 적용

## [1.0.3] - 2026-04-15
- App.tsx 파일 크기 최적화를 위한 UI 컴포넌트(LogItem, LogImage, LogAvatar 등) 및 유틸리티 함수 분리 리팩토링

## [1.0.1] - 2026-04-13
