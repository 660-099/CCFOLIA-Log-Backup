# Changelog

## [1.2.2] - 2026-04-23
- [Added] 로그 필터 최상단 '고정형 모드(상단 고정 바)' 추가 및 [사용 안 함 / 플로팅 / 고정형] 3단계 필터바 모드 선택 기능 도입
- [Changed] 나레이션 필터와 원본 캐릭터 필터를 분리하여 독립적인 두 개의 체크박스로 개별 제어할 수 있도록 세분화
- [Changed] 로그 필터 컨트롤러 메뉴 위치 조정(CSS 출력 형식 위) 및 디자인 전체 통일성(딤드 화이트, 12px 등) 반영
- [Fixed] 캐릭터 라이브러리 및 툴팁 호버 시 좌표 계산 시차로 인해 좌측 상단에서 깜빡이며 나타나는 렌더링(Flickering) 현상 해결하기 위해 초기 로딩 최적화

## [1.2.1] - 2026-04-23
- [Added] 나레이션 캐릭터를 필터바 목록(CHARS) 최상단에 고정 표시하고 "나레이션"으로 이름 변경 적용
- [Optimized] HTML 내보내기 시 코드 효율성을 위해 CSS 중복 제거, HTML 구조 간소화 및 자바스크립트 화살표 함수 도입
- [Fixed] 로그 필터바가 꺼진 항목만 숨기도록 Blacklist 로직 수정 및 누락되는 항목 없도록 구조 보강 시각적 디자인 유지 

## [1.2.0] - 2026-04-21
- [Added] 캐릭터 보관함(라이브러리)
- [Added] 테마 설정에 다크 모드 및 라이트 모드용 배경색 선택 기능 추가 적용
- [Optimized] 전체 구조의 안정성 향상을 위해 포탈(Portal) 방식의 드롭다운 UI 적용 (캐릭터 보관함 - 미리보기 화면 잘림 증상 해소)

## [1.1.2] - 2026-04-19
- [Added] 자동 파싱 시 HTML 파일의 요소 내 속성을 분석(속성이 없으면 이름 기준)하여 각 탭 및 캐릭터에 고유 ID(tabId, charId)를 부여해 관리하는 체계 구축
- [Added] 섹션 관리 및 스크롤 점프 버튼 메뉴 기능 도입
- [Fixed] 내보내기(Export) HTML 코드 디자인(인라인 및 내부 스타일 포함) 수정
- [Added] 프로젝트 파일(.json) 저장 및 불러오기 시 편집 내역(로그 내용 수정/삭제)이 모두 유지되도록 데이터 포함 기능 추가
- [Changed] 프로젝트 파일 생성 시 우선적으로 사용자가 편집한 '프로젝트 제목'으로 파일명(Title)이 지정되도록 변경
- [Added] 정보/가리기(숨김) 처리 된 탭 설정에서도 추가된 삽화(이미지)는 로그에서 사라지지 않고 독립적으로 유지 및 출력되도록 개선
- [Fixed] 특정 탭이나 캐릭터의 가시성을 변경할 때 간헐적으로 발생하던 리액트 훅 에러 해결 및 렌더링 안정성 향상

## [1.1.1] - 2026-04-18
- [Added] 브라우저 웹 스토리지(LocalStorage)를 활용한 '설정 기억하기' 기능 및 토글 스위치 도입 (새로고침 후에도 테마 및 UI 설정 유지)
- [Optimized] 다양한 환경의 HTML 로그를 원활하게 지원하기 위해 파싱(parser.ts) 로직 유연화 (특정 구조에 의존하지 않고 모든 요소의 color 속성 탐지)
      (https://lispcoc.github.io/ccfolia_log_getter/ 에서 추출한 로그도 사용 가능)
- [Optimized] 파일 업로드 UI 직관성 개선 (업로드한 원본 파일명 표시, 즉각적인 컬러 하이라이팅 피드백, 업로드 취소(X) 기능 추가)
- [Fixed] 밝은(Light) 테마 사용 시 배경색과 겹쳐 보이지 않던 일부 안내 텍스트 렌더링 버그 수정
- [Changed] 파일 업로드 대기 심볼의 펄스(Pulse) 애니메이션 제거
- [Fixed] 커스텀 툴팁에 마우스를 올릴 때 너무 빠르게 사라지던 현상을 방지하기 위해 툴팁 주변에 투명 가상 영역 버퍼 적용
