# 코코포리아 로그 백업 (CCFOLIA Log Backup)

CCFOLIA에서 추출한 TRPG 로그를 가독성 좋게 정리하고 백업하는 도구입니다.

## 요약 (Summary)
이 도구는 코코포리아 HTML 로그를 파싱하여 캐릭터별 색상, 탭 필터링, 이미지 삽입, 로그 분할 등 다양한 편집 기능을 제공하며, 최종적으로 정돈된 HTML 파일로 내보낼 수 있는 올인원 로그 포매터입니다.

## 주요 기능
- **로그 업로드**: CCFOLIA HTML 로그 파일을 업로드하여 즉시 변환 (상단 빈 로그 자동 제거)
- **캐릭터 설정**: 이름, 색상, 아바타 이미지 설정 및 가시성 조절 (이름 호버 시 툴팁 표시)
- **탭 설정**: 탭별 출력 형식(메인, 보조, 정보, 비밀 등) 및 색상 설정
- **미리보기 편집**: 로그 내용 실시간 수정, 삭제, 이미지 삽입 (URL 기반, 크기/정렬 조절)
- **로그 분할**: 원하는 지점에서 로그를 분할하여 섹션별 제목 설정 및 관리
- **내보내기**: 전체 또는 섹션별 HTML 다운로드, 클립보드 복사, ZIP 일괄 저장
- **스타일 관리**: 설정한 모든 스타일과 편집 내용을 JSON 파일로 저장 및 불러오기
- **사용자 편의**: 되돌리기(Undo)/다시 실행(Redo), 다크/라이트 모드, 폰트 설정 지원
- **이미지 배경 상자 숨김**: 캐릭터 아바타의 배경 상자를 투명하게 처리하여 깔끔한 레이아웃 제공
- **탭별 통합**: 동일한 탭의 연속된 로그를 하나의 블록으로 시각적으로 연결하여 가독성 향상

## 사용 방법
1. **로그 업로드**: `파일` 탭에서 코코포리아에서 추출한 HTML 파일을 업로드합니다.
2. **스타일 설정**: `탭` 및 `캐릭터` 탭에서 각 항목의 색상, 아바타, 가시성을 설정합니다.
3. **로그 편집**: 미리보기 화면에서 로그 내용을 수정하거나 삭제하고, 필요한 위치에 이미지를 삽입합니다.
4. **섹션 분할**: 긴 로그의 경우 `여기서 분할` 버튼을 눌러 섹션을 나누고 이름을 지정합니다.
5. **저장 및 내보내기**: `내보내기` 버튼을 통해 HTML 파일이나 ZIP 파일로 저장합니다.

## 기술 스택
- **Frontend**: React, TypeScript, Vite
- **Styling**: Tailwind CSS, Lucide React (Icons)
- **Animation**: Framer Motion
- **Utilities**: JSZip (ZIP export), React Colorful (Color picker)
